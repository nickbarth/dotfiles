# dotfiles

An archive of a few personal dotfiles for easy access.

### Dependencies

```terminal
OSX/BREW: git cmake tmux stow
LINUX/APTGET: git make tmux stow
```

### Setup

```terminal
git clone --recursive git@github.com:nickbarth/dotfiles.git && cd dotfiles
stow -v2 .
```
